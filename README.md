## 概要
いくつかの単語を入力することで，それらの単語から推測される絵文字を提供する web アプリケーションです．
PKSHA Technology の日本語テキスト埋め込みモデル [GLuCoSE](https://huggingface.co/pkshatech/GLuCoSE-base-ja) に絵文字に関する情報を埋め込んだモデルを利用しています．
